O Bitrix24 é um software de mercado amplamente conhecido que possui funcionalidades nativas e abertura para customizações, integrações e automações.

## Customização CR Sistemas

A CR Sistemas customizou o CRM para atender às necessidades dos clientes que utilizam o ERP **CRIS**. Isso inclui:

- Criação de uma **pipeline** personalizada
- Parametrização integrada com o CRIS
- Campos personalizados para mapear a jornada do cliente

<Warning>
  Nossa companhia do Bitrix24 é **compartilhada entre os clientes da CR Sistemas**, e prezamos muito pela organização dos dados.

  Qualquer automação ou integração que fuja do escopo abaixo — mesmo que possua endpoint nativo do Bitrix24 — deve ser **previamente alinhada com o time CR Sistemas** para evitar transtornos, retrabalhos e atrasos desnecessários.
</Warning>
