## Token API

<Check>
  **Status: Disponível**
</Check>

Por razões de segurança, a **criação de tokens para a API do Bitrix24 é feita exclusivamente pelo time CR Sistemas**.

Para solicitar um token, entre em contato com a equipe. 😊

<Info>
  O token gerado é vinculado à sua unidade e respeita as permissões configuradas para o seu perfil de acesso no Bitrix24.
</Info>
