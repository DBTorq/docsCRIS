## Autenticação CRIS

A maioria dos endpoints do CRIS exige um token de autenticação gerado pelo endpoint [`loginApi`](/cris/entrada/apoio#login-api).

```http
Authorization: {token_gerado_pelo_loginApi}
Content-Type: application/json
```

<Note>
  O token deve ser renovado periodicamente. Consulte o endpoint [loginApi](/cris/entrada/apoio#login-api) para mais detalhes.
</Note>

## Status dos Endpoints

| Status | Descrição |
|---|---|
| ✅ **Disponível** | Endpoint funcional e pronto para uso |
| 🔜 **Melhoria futura** | Indisponível no momento, previsto para versões futuras |
